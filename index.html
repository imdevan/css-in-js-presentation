What is CSS-In-JS

Really any library that puts styles in JS…

CSS-In-JS attempts toswww put the best of the css pre-processors on the same build pipeline as the rest of your React/Angular/Vue code to give you the developer a better experience and while improving build performance

Who is it for?

Anyone who is already developing with React, Angular, or Vue. That want to improve the performance and development experience of styling their apps.

What makes her so special?

  • Best of CSS Modules and inline styles
  • Backwords compatible with inline-styles
  • Pages styles load really really fast
  • Super Simple Server Side Rendering
  • Automagical style caching
  • Works with React, React Native, and even React VR!
